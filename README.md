🛠️ How to Use
Open the file in your preferred editor.
Set the input directory to the texture/resource pack you want to recolor.
Set the output directory to where you want the newly recolored pack to be saved.
(Optional) Customize colors:
You can assign individual colors to specific block types or groups.
Or simply keep the default color configuration provided by the tool.
Run the tool and your recolored pack will be generated in the output folder.
