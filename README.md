## 🛠️ How to Use

1. Open the file in your preferred editor.

2. Set the **input directory** to the texture/resource pack you want to recolor.

3. Set the **output directory** to where you want the newly recolored pack to be saved.

4. *(Optional)* Customize colors:

   * You can assign individual colors to specific block types or groups.
   * Or simply keep the default color configuration provided by the tool.

5. Run the tool and your recolored pack will be generated in the output folder.
